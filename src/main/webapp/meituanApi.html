1、用户登录接口

 

url：/api/meituan/login

method：get

 

请求参数：

字段

类型

是否必须

备注

userName

string

必须

用户登录的用户名

password

string

必须

用户用于登录的密码

appkey

string

必须

用户访问接口时所必须携带的身份凭据

 

返回结果：

登录成功返回：

{

"msg":"登录成功",

"status":"success"

}

登录失败返回情况：

1）用户名为空时登录失败返回值：

{

"msg":"找不到用户名",

"status":"fail"

}

2）密码为空时登录失败返回值：{“status”:“fail”}  {“massage”:“The password is null”}

{

"msg":"密码错误",

"status":"fail"

}

3）用户名不符合格式登录时失败返回值：

{

"msg":"用户名必须为4-16位的字母数字下划线组成",

"status":"fail"

}

4）密码不符合格式登录时失败返回值：

{

"msg":"密码必须为6-16位的字母数字下划线组成",

"status":"fail"

}
