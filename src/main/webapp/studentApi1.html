3.删除一个学生信息

url：/api/student/delBySno

method :get

请求参数：

字段

类型

是否必须

备注

appkey

String

必须

用户访问接口时所必须携带的身份凭据

sNo

String

必须

该学生的学号

返回结果：

1.删除成功时：

{

"msg":"删除成功",

"status":"success"

}

2.删除失败时：

{

  "msg": "未找到该学生",

  "status": "fail"

}

或

{

  "msg": "发生未知错误",

  "status": "fail"

}

 

4.添加一个学生的信息

 

url:/api/student/addStudent

method:get

请求参数：

字段

类型

是否必须

备注

appkey

String

必须

用户访问接口时所必须携带的身份凭据

sNo

String

必须

该学生的学号

name

String

必须

学生姓名

sex

int

必须

学生性别“0”是男，“1”是女（默认是0）

birth

int

必须

学生出生年份(默认是0)

phone

String

必须

学生的手机号

address

String

必须

学生的住址

email

String

必须

学生邮箱

返回结果：

1.成功返回：

{

"msg":"添加成功",

"status":"success"

}

2.失败返回：

1)学生已存在：

{

"msg":"该学生已经存在",

"status":"fail"

}

2)XXX为格式错误：

   如：学号的信息格式错误：

{

"msg":"学号必须为4-16位的数字组成",

"status":"fail"

}
